Геометрические фигуры: Треугольник

Данный репозиторий создан в рамках учебной работы по изучению платформы GitHub и практическому использованию системы контроля версий Git. Здесь реализованы функции для работы с геометрической фигурой «Треугольник»: вычисление периметра, площади по формуле Герона и проверка на равнобедренность.
Цели и задачи работы

    Освоить базовые принципы работы с Git (коммиты, ветки, слияния, теги).

    Научиться использовать GitHub для хранения, обновления и распространения исходного кода.

    Получить практические навыки оформления репозитория (README.md, лицензия, структура).

    Реализовать алгоритмы для заданной геометрической фигуры в виде набора функций.

Задание

Разработать алгоритм решения задачи по индивидуальному заданию:

Фигура: Треугольник
Функции:

    Вычисление периметра.

    Вычисление площади по формуле Герона.

    Проверка на равнобедренность.
